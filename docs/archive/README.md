# Archive

This directory contains historical pass logs and development-era notes. These
files are kept for traceability, but current workflow instructions live in:

- `README.md`
- `docs/testing.md`
- `docs/release/RELEASE_CHECKLIST.md`

Root-level `README_PASS*.txt` history is archived under
`docs/archive/readme_passes/`.
Long-form documents pruned from the active docs directory are archived under
`docs/archive/active_doc_history/`.

Do not use archived pass logs as the source of truth for current test or
release workflow.
